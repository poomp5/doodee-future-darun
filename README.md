# Doodee Future

> A digital platform that helps Thai students plan and prepare for university admission in a structured and personalized way.

Doodee Future is a web platform designed to help students organize their university preparation journey in one place. Instead of searching through multiple websites for exam preparation, portfolio examples, scholarships, and activities, students can access all essential tools through a single structured system.

The platform transforms scattered educational information into a **clear preparation pathway for each student**.

---

# Overview

Every year, hundreds of thousands of Thai students compete for university seats through Thailand’s centralized admission system. However, the biggest challenge students face is not only competition - but the **lack of a structured preparation system**.

Students often need to gather information from many disconnected sources such as:

- Portfolio examples on social media
- Tutor websites and exam preparation resources
- University announcements
- Advice from seniors

This fragmented ecosystem leads to confusion, wasted time, and stress.

**Doodee Future solves this by organizing the entire preparation process into one integrated platform.**

# Platform Architecture

![Doodee Future Tech Stack](https://cdn.pranakorn.dev/doodee/063d94bd-e427-4464-a0ef-136eb9e5ed2f.png)

The system architecture is divided into four layers:

### Frontend Layer
- Next.js
- Tailwind CSS

### Backend Layer
- Next.js API Routes
- Prisma ORM

### AI / Processing Layer
- Tesseract.js (OCR)
- PDF.js (document processing)
- Typhoon LLM
- DeepSeek (fallback model)

Typhoon is a Thai-focused large language model designed for strong Thai language performance and efficient real-world deployment. :contentReference[oaicite:0]{index=0}

### Infrastructure Layer
- Linux VPS
- Docker Containers
- Neon PostgreSQL (serverless database)

# Core Features

## AI Analyzer

An AI-powered system that analyzes student profiles including:

- Academic performance
- Activities and achievements
- Portfolio content

The system provides insights such as:

- Strengths and weaknesses
- Areas for improvement
- Strategic guidance for university admission.

---

## Mock Exam & Performance Analytics

Students can take simulated exams and receive automatic performance analysis to evaluate their readiness for real university entrance exams.

Features include:

- Mock exams
- Score analytics
- AI performance analysis
- Personalized improvement suggestions

---

## Free Learning Resources

A community-driven hub where students can access:

- Study summaries
- Past exam papers
- Shared learning materials

This helps reduce inequality in access to educational resources.

---

## Community & Opportunity Network

Students can explore:

- Real portfolio examples from seniors
- Academic competitions
- Exchange programs
- Scholarships

The platform also enables students to connect and exchange knowledge.

# Competitive Advantage

While several platforms in Thailand offer **partial services** such as exam preparation or student communities, no platform integrates all elements of university preparation into one ecosystem.

Doodee Future integrates:

- Portfolio planning
- Exam preparation
- Opportunity discovery
- Community learning
- AI-driven analysis

As user data grows, the platform gains a **data network effect**, allowing increasingly accurate personalized recommendations.

# Business Model

Doodee Future operates using a **B2C + B2B hybrid model**.

### B2C (Students)
Students can use the platform **for free**.

### B2B (Institutions)

Revenue comes from educational institutions through:

- University & Institution Subscription
- AI Matching for student recruitment
- Educational data analytics
- Program promotion

Additional revenue streams:

- Course marketplace commission (10–15%)
- Sponsored competitions or programs
- Data insights packages for institutions
